# pnl-summary

## 功能定位

查詢指定時間區間內的總損益總結（Profit & Loss Summary）。此命令將自動聚合計實現在特定起訖日內的各筆交易，給予使用者最巨觀的損益全貌，省去逐筆加總的麻煩。

## 適用情境

- **月報/週報核對**：定期（如月底、週五）追蹤整體投資組合或當沖策略的績效成績。
- **整體績效檢視**：一眼看穿一段時間內的賺賠比、總收益、與相對成效。

## 命令語法

```powershell
python shioaji_cli.py accounts pnl-summary [參數選項]
```

## 參數詳解

| 參數 | 必填 | 預設值 | 可選值 | 說明 |
|---|---|---|---|---|
| `--start` | 是 | 無 | `YYYY-MM-DD` | 查詢統計區間的起始日期。 |
| `--end` | 是 | 無 | `YYYY-MM-DD` | 查詢統計區間的結束日期。 |
| `--account-kind` | 否 | `any` | `auto / any / stock / futures` | 帳戶種類。可用來篩選特定市場的損益。 |
| `--account-id` | 否 | 當前預設帳戶 | 有效的帳號字串 | 若要查詢特定帳戶的總結績效請提供。 |

## 核心回傳資料結構

執行完畢後會返回 `ProfitLossSummaryTotal` 結構及其內部明細，常見欄位涵蓋：

- **code**: 商品/合約代碼。
- **quantity**: 區間內累計平倉或結算的數量。
- **entry_price** / **cover_price**: 平均進場價與平均回補（平倉）價。
- **pnl**: （損益）區間內實際結算之絕對賺賠金額。
- **pr_ratio**: 報酬率（即盈虧比例）。

## 實戰範例

**範例 1：查詢 2026年3月份整月證券總損益**

```powershell
python shioaji_cli.py accounts pnl-summary --account-kind stock --start 2026-03-01 --end 2026-03-31
```
*說明：拉出整個三月份在台股現貨與融資券帳戶下的累計獲利與虧損總計。*

## 注意事項與雷區

1. **起訖日限制**：
   由於 `start` 與 `end` 區間跨度可能影響券商主機的回應效率，若發現查詢太慢或 TimeOut，可嘗試將區間縮短（例如降為一周）。
2. **僅限已實現損益**：
   此清單與總結主要聚焦在「已平倉/結算了結」的已實現損益。帳上現有持倉的波動（未實現損益）通常需要透過 `positions` 以即時市價估算。
