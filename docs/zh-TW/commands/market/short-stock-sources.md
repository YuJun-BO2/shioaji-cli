# market short-stock-sources

## 功能定位
查詢特定標的券源數量（即放空可借券的庫存數量），當打算進行無券放空或當沖先賣出時必備的查詢命令。

## 適用情境
- 當沖交易先賣後買前，確認系統是否仍有足夠的券源配額
- 欲留倉建立空頭部位時的前置庫存檢查
- 結合策略腳本動態判定這檔標的是否「有券可空」

## 命令語法
```powershell
python scripts/shioaji_cli.py market short-stock-sources --codes <股票代號1> [<股票代號2> ...]
```

## 參數詳解
| 參數 | 必填 | 預設值 | 可選值 | 說明 |
|---|---|---|---|---|
| `--codes` | 是 | 無 | 不限 | 要查詢券源的股票代號列表，可用空格分隔多個代號 |

## 核心回傳資料結構
回傳的清單包含各股券源資訊：
- `code`: 股票代號
- `short_stock_source`: 可用券源數量（張數）
- `ts`: 更新時間戳記或日期戳記

## 實戰範例
查詢台積電 (2330) 與鴻海 (2317) 的券源庫存：
```powershell
python scripts/shioaji_cli.py market short-stock-sources --codes 2330 2317
```

## 注意事項與雷區
- **高度變動性**：券源由券商系統即時配發且全體客戶共用，查詢有券不代表下單時一定搶得到，建議查到後盡快執行決策。
- **僅代表查詢券商庫存**：此數量僅反映當前交易帳戶所屬券商可提供的券源，不同券源體系狀態可能不同。
- **無效代碼回報**：對無法放空的 ETF 或特定商品查詢，可能回傳空值或 0。
