# market notice

## 功能定位
查詢市場公告的「注意股」名單，取得近期價格波動過大或交易情況異常而被證交所列入警示的商品資訊。

## 適用情境
- 執行交易前，檢查目標股票是否因波動劇烈而被列為注意股
- 作為選股策略的過濾條件，避免因注意股的異常波動或潛在處置風險而造成損失
- 風控系統每日例行檢查與警示

## 命令語法
```powershell
python scripts/shioaji_cli.py market notice
```

## 參數詳解
此命令無需額外參數，預設返回全市場所有注意股清單。
| 參數 | 必填 | 預設值 | 可選值 | 說明 |
|---|---|---|---|---|
| 無 | | | | |

## 核心回傳資料結構
以 JSON 陣列返回名單，每筆記錄代表一檔注意股：
- `code`: 個股代碼
- `announced_date`: 公告日期
- `reason`: 列入注意股的具體原因描述
- `close`: 參考收盤價
- `updated_at`: 資料更新時間戳記

## 實戰範例
直接獲取今日所有注意股：
```powershell
python scripts/shioaji_cli.py market notice
```

## 注意事項與雷區
- **潛在處置風險**：連續多日被列為注意股的標的，極高機率被列入「處置股」，請特別留意流動性及委託限制變化。
- **交易所公告時間**：注意股名單通常於每日收盤後由證券交易所公告，建議盤後或隔日盤前查詢以獲得最正確資訊。
